# Hướng dẫn thêm file âm thanh

Để sử dụng đầy đủ tính năng âm thanh thông báo, bạn cần thêm các file MP3 sau vào thư mục này:

## Files cần thiết:

1. **notification.mp3** - Âm thanh khi nhận tin nhắn/thông báo
2. **sent.mp3** - Âm thanh khi gửi tin nhắn
3. **ring.mp3** - Âm thanh chuông cuộc gọi
4. **end.mp3** - Âm thanh kết thúc cuộc gọi
5. **accepted.mp3** - Âm thanh chấp nhận cuộc gọi
6. **typing.mp3** - Âm thanh typing
7. **success.mp3** - Âm thanh thành công
8. **error.mp3** - Âm thanh lỗi

## Nguồn tải âm thanh miễn phí:

- **Pixabay**: https://pixabay.com/sound-effects/
- **Freesound**: https://freesound.org/
- **Notification Sounds**: https://notificationsounds.com/
- **Zapsplat**: https://www.zapsplat.com/

## Lưu ý:
- File nên có dung lượng nhỏ (< 100KB)
- Định dạng MP3
- Âm lượng vừa phải (chuẩn hóa trước khi thêm)
